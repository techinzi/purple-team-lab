# VMware Workstation Pro - Download & Installation Guide

## Overview
VMware Workstation Pro is the virtualization platform.
It hosts all virtual machines including the Windows 10 attack target, Ubuntu Server
(Wazuh + Caldera), and Kali Linux attacker machine.

- **Version:** 17.6.4
- **Host OS:** Windows 11
- **Cost:** Free for personal use
- **Download Source:** Broadcom Support Portal

---

## Part 1 - Download

### Step 1 - Create a Broadcom Account
Browse to the [Broadcom Support Portal](https://support.broadcom.com/) and register with a valid email address.

### Step 2 - Navigate to Free Software Downloads
After logging in, go to: `Software > Cyber Security Software > My Downloads`

![Broadcom Home Page](/images/lab-setup/vmware/01-my-downloads.png)

Click on **"Free Software Downloads available HERE"**:

![Downoad Section](/images/lab-setup/vmware/02-free-downloads.png)

### Step 3 - Search for VMware Workstation Pro
In the search bar, type: **vmware workstation**.
Click on **VMware Workstation Pro** in the search results.

![Search Section](/images/lab-setup/vmware/03-search-vmware.png)

### Step 4 - Select Windows Version
Click on the product link for **Windows**.

![Select OS](/images/lab-setup/vmware/04-vmware-windows.png)

### Step 5 - Select Latest Version
Select the latest available version to proceed.

![Select Version](/images/lab-setup/vmware/05-latest-release.png)

### Step 6 - Download
Click the **Download** button.

![Download](/images/lab-setup/vmware/06-download-button.png)

## Part 2 - Installation

### Step 1 - Run the Installer
Locate the downloaded `.exe` file and double-click to launch the installer.
Follow the installation wizard using the default options at each step and complete the installation.

### Step 2 - Verify Installation
After installation completes, open VMware Workstation Pro from the Start Menu or Desktop shortcut.
The home screen should appear with no license prompt.

Navigate to `Help > About VMware Workstation` and confirm the version

![Verify Installation](/images/lab-setup/vmware/07-vmware-version-check.png)
